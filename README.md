# Gro-Cart

Gro-Cart is a comprehensive Grocery Store Management Application developed using Flask, designed to streamline the operations of a grocery store.

## Features

- **Product Management**: Create, update, and delete product categories and individual products.
- **Automated Stock Management**: Real-time stock updates and expiration management for efficient inventory control.
- **Sales Summary**: Graphical representation of overall sales and category-wise sales for insightful business analysis.
- **User-Friendly Customer Interface**: Convenient purchase of multiple products with real-time price calculation and itemized billing.

## Getting Started

1. **Installation**: Clone the repository and install dependencies.

    ```bash
    git clone https://github.com/your-username/gro-cart.git
    cd gro-cart
    pip install -r requirements.txt
    ```

2. **Run the Application**:

    ```bash
    python app.py
    ```

3. **Access the Application**:

    Open your web browser and navigate to [http://localhost:5000](http://localhost:5000).

## Technology Stack

- **Backend**: Flask (Python)
- **Database**: SQLAlchemy (SQL)
- **User Interface**: HTML, CSS, Bootstrap

## Project Structure

- `/templates`: HTML templates
- `/static`: Static assets (CSS, JavaScript)
- `/app`: Flask application files
